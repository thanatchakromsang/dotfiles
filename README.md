# Dotfiles

My universal dotfiles for every unix machine (Linux, macOS)

- neovim
- tmux configs
- zsh config + alias
- i3wm configs (polybar, dunst, compton, xresources)
- etc.

## __Installation__

```
    sudo pacman -S git

    git clone https://github.com/thanatchakromsang/dotfiles.git ~/.dotfiles && cd ~/.dotfiles && ./setup.sh
```
